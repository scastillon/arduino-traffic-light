# Arduino Traffic Light Simulation

A 3-phase traffic light control system built for Arduino Uno using C++ logic.

## 📌 Project Overview
This project simulates an automated traffic light system controlling time cycles for Red, Yellow, and Green LEDs through Digital Pins on an Arduino Uno.

## ⚙️ Circuit Specs & Pin Distribution
- **Green LED**: Pin 11 (Active: 5s)
- **Yellow LED**: Pin 12 (Active: 2s)
- **Red LED**: Pin 13 (Active: 5s)
- **Current Protection**: 220Ω resistors connected in series with each LED cathode to GND.

## 🛠️ Simulation Platform
Designed, wired, and verified in Tinkercad Circuits.
